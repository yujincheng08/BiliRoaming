# 支持以下功能

- 解除B站番剧区域限制
- 港澳台CDN加速
- 缓存番剧
- 支持国际版和概念版
- 自定义主题色
- 关闭青少年模式弹窗
- 显示评论区楼层
- 概念版添加直播入口
- 不以小程序形式分享
- 自动点赞视频

# Features

- Unlock bangumi area limit
- CDN speedup
- Download bangumi
- Support International ver and blue ver
- Customize theme
- Close teenager dialog
- Show comment floor
- Add live entry to blue ver
- Share without mini programs
- Like videos automatically
